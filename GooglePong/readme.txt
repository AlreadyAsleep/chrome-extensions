this extension is a simple pong clone to run as a Google Chrome extension

to run this extension:
 1)go to chrome://extensions
 2)select developer mode in top right corner
 3)select 'load unpacked extension'
 4)select the app folder in the GooglePong dir
 5)the extension should show up with the icon in the top-right corner of the browser
 6)select the extension and the use arrow keys to move 
 
 enjoy
